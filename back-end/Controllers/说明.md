**处理 HTTP 请求的主要入口**，此文件夹中存储：**负责接收前端 HTTP 请求，调用 Service 层，返回 JSON 结果**的C#文件。属于前端和后端的接口。
| 方法                  | 说明            |
| ------------------- | ------------- |
| `Ok(object)`        | 返回 200 + 数据   |
| `CreatedAtAction()` | 返回 201 + 路由信息 |
| `NotFound()`        | 返回 404        |
| `BadRequest()`      | 返回 400        |
| `NoContent()`       | 返回 204（无内容）   |
