# 首页接口补充说明文档

### 调用接口

**1. 用户简介接口**

- 用来获得用户的介绍信息

- 输入表单说明

| 字段名   | 类型   | 是否必填 | 说明 |
| -------- | ------ | -------- | ----- |
| userId | int | 是 | 用户编号 |

- 获得信息说明

| 字段名       | 类型        | 说明           |
| ------------ | ----------- | -------------- |
| name         | string      | 用户昵称       |
| phoneNumber  | number      | 用户电话号码   |
| image        | string      | 用户头像 URL   |

- 接口地址: `GET /api/user/profile/userProfile`

**2. 用户地址接口**

- 用来获得用户的地址信息

- 输入表单说明

| 字段名   | 类型   | 是否必填 | 说明 |
| -------- | ------ | -------- | ----- |
| userId | int | 是 | 用户编号 |

- 获得信息说明

| 字段名       | 类型        | 说明           |
| ------------ | ----------- | -------------- |
| name         | string      | 收货人姓名     |
| phoneNumber  | number      | 收货人电话号码 |
| address      | string      | 收货地址       |

- 接口地址: `GET /api/user/profile/address`
